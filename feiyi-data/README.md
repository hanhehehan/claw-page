# 非遗餐饮门店榜单 - 数据导出包

## 项目简介

2026年中国菜申报世界非物质文化遗产参考资料库，覆盖全国30个城市、1367条非遗餐饮门店记录。

## 文件说明

| 文件 | 说明 |
|------|------|
| `非遗餐饮门店数据.json` | 结构化JSON数据，可直接导入程序使用 |
| `非遗餐饮门店数据.csv` | CSV格式，可用Excel/WPS直接打开 |
| `非遗餐饮门店榜单.html` | 完整可视化网页（含数据+代码），浏览器直接打开 |
| `README.md` | 本说明文件 |

## 数据字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `城市` | string | 门店所在城市（30个城市） |
| `门店名` | string | 门店名称 |
| `非遗项目` | string | 关联的非物质文化遗产项目名称 |
| `级别` | string | 非遗级别：国家级/省级/市级/区级 |
| `类型` | string | 非遗类型：传统技艺/传统医药/民俗等 |
| `备注` | string | 补充说明（创始年份、批次等） |
| `来源URL` | string | 数据来源链接 |
| `来源名称` | string | 数据来源名称 |
| `可信度` | string | A=官方可验证 / B=名录可追溯 / C=待核实 |
| `验证说明` | string | 验证方式说明 |
| `poi_id` | string | 大众点评POI ID（部分门店已匹配） |
| `dp_url` | string | 大众点评门店链接（部分门店已匹配） |

## 数据统计

- 总记录数：1367条
- 覆盖城市：30个
- 已匹配POI：96家独立门店（137条记录）
- 可信度分布：A级（官方可验证）为主

## 如何使用

### Python 导入
```python
import json
with open('非遗餐饮门店数据.json', 'r', encoding='utf-8') as f:
    data = json.load(f)
# data 是一个列表，每个元素是一个字典
print(f"共 {len(data)} 条记录")
```

### JavaScript 导入
```javascript
fetch('非遗餐饮门店数据.json')
  .then(r => r.json())
  .then(data => {
    console.log(`共 ${data.length} 条记录`);
  });
```

### Excel / WPS
直接双击打开 `非遗餐饮门店数据.csv` 即可。
