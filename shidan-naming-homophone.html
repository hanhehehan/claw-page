「食单」命名 · 实体谐音版

核心思路：找到一个真实的动植物/水果，它的名字里天然带着一个跟"收集"同音或近音的字。名字本身是实体词（人人认识），但谐音赋予了收集含义。闲鱼=咸鱼改字，蚂蚁森林=蚂蚁+森林，石榴=拾留，仓鼠=藏鼠——组合创造含义。

命名公式：实体词（动植物/水果，自带认知）+ 谐音字（收集意向：拾/留/藏/囤/录/夹/聚/满/单/集/备）= 好记 + 年轻 + 美食 + 收集
关键原则：不直接拿动植物名字硬贴，而是利用谐音让实体名本身就讲出"收集美食"的故事。石榴=拾留（拾起+留存），不需要解释，听到就知道是水果，稍微一想就懂收集含义。

01 拾留类 石(shí)=拾 · 榴(liú)=留 → 拾留=拾起+留存
拾=捡起来收集，留=留住保存。这两个字合在一起就是收集的完整动作。找到一个实体词，名字里同时藏着"拾"和"留"的谐音——石榴。而且石=食，一音三义：石(石头/实体)、拾(收集)、食(美食)。
石榴 全场首推 石(=拾/食) + 榴(=留) = 拾留/食留 shí liú
两个字全是精确谐音，水果本身就是收集容器。石(shí)=拾(shí)=食(shí)，一音三义；榴(liú)=留(liú)。石榴=拾留=食留=拾起留住=吃过的留住。更妙的是，一颗石榴掰开来，里面密密麻麻几百颗籽，全兜在一个壳里——它本身就是"收集容器"的终极隐喻。每颗籽=你收录的一家店，整个石榴=你的食单。石榴汁、石榴籽是年轻人饮品甜品常见元素，不old。
画面感：掰开一颗石榴，红宝石般的籽密密麻麻排列，每颗籽是一家你验证过的好店
两字精确谐音（拾+留）；水果本身=收集容器（百籽聚一壳）；石还=食（三重谐音）；红色热烈视觉强；Z世代饮品常见元素；品牌延展自然（石榴籽=每家店、石榴红了=新收录）
石榴在部分人认知中偏传统水果；需要品牌化重新定义年轻感

画面感：山间岩石上采石耳
本身就是山珍美食；拾+食双谐音

本类精选：石榴。这是全场最满意的名字——两个字全是精确谐音（拾+留），水果本身就是收集容器（百籽聚一壳），石还=食，一音三义。跟闲鱼是同一个级别的命名逻辑：咸鱼→闲鱼改一个字，石榴→拾留两个字全谐音，而且实体本身就是收集容器。
02 藏类 仓/苍(cāng) ≈ 藏(cáng) → 收藏储存
藏=收藏、储存、藏好。仓(cāng)和苍(cāng)跟藏(cáng)只差一个声调，近音。而仓本身就有"粮仓、仓库"的意思——天然就是"收集储存食物的地方"。仓鼠=藏鼠=会藏东西的鼠。
仓鼠 类最佳 仓(cāng≈藏cáng) + 鼠 = 藏鼠 cāng shǔ
仓=粮仓=收集储存食物的地方，仓鼠天生就在收集食物。仓(cāng)≈藏(cáng)，仓鼠=藏鼠=会藏东西的鼠。仓鼠最标志性的行为是什么？腮帮子塞满食物搬回窝里存着。这个行为天然就是"收集+储存"。你的食单=你的仓鼠窝，每收录一家店=腮帮子又鼓了一点。仓本身就不是白叫的——仓=粮仓=仓库，是"收集储存粮食的场所"，含义比谐音还直接。
画面感：一只小仓鼠腮帮子鼓鼓的，蹦蹦跳跳往窝里搬好东西
仓=粮仓=收集场所（含义比谐音更直接）；仓鼠行为=收集（腮帮子塞满）；形象极萌IP化强；"仓鼠窝"=你的食单；"腮帮子越来越鼓"=收录越来越多
仓鼠已有宠物品牌联想；近音非精确谐音（cāng vs cáng）
苍耳 苍(cāng≈藏cáng) + 耳 = 藏耳 cāng ěr
苍(≈藏)+耳。苍耳是一种植物，果实带刺会粘在衣服上跟着你走——"粘住不放=收集留住"。藏耳=藏在耳朵旁=粘着你走的好店。苍耳的"粘附"特性=收集行为。
画面感：走过草丛，苍耳粘了一裤腿，甩都甩不掉——好店粘住你不放
苍耳粘附=收集的物理隐喻独特；植物实体
苍耳知名度一般；粘附感可能让人不舒服
苍鹭 苍(cāng≈藏cáng) + 鹭(=录lù) = 藏录 cāng lù
苍(≈藏)+鹭(=录)。苍鹭=藏录=收藏记录。一个名字里同时藏了两个收集含义的字！苍=藏（收藏），鹭=录（记录），组合起来就是"收藏并记录"=食单的核心功能。
画面感：一只苍鹭站在水边一动不动等待猎物——耐心守候好店
双谐音（藏+录）罕见；鸟形象优雅
苍鹭知名度一般；形象偏冷不够亲切
仓鸮 仓(cāng≈藏cáng) + 鸮(xiāo) = 藏鸮 cāng xiāo
仓(≈藏)+鸮。仓鸮=猫头鹰的一种。猫头鹰夜间出动=你在夜间整理食单。仓=仓库=粮仓=收集粮食的地方。
画面感：夜晚仓鸮站在粮仓上守着收集来的粮食
仓=粮仓含义直接；猫头鹰有夜间属性
鸮字生僻；猫头鹰部分文化中不吉
苍鹰 苍(cāng≈藏cáng) + 鹰 = 藏鹰 cāng yīng
苍(≈藏)+鹰。苍鹰=藏鹰=会藏的鹰。鹰=视野广阔=你遍览全城美食。
画面感：苍鹰在高空盘旋，锐眼锁定美食

画面感：春天黄鹂在枝头歌唱，万物更新

苍兰 苍(cāng≈藏cáng) + 兰 = 藏兰 cāng lán
苍(≈藏)+兰。苍兰=藏兰=收藏兰花=收藏美好事物。苍兰是一种花，清雅芬芳。

本类精选：仓鼠。仓=粮仓=收集储存粮食的场所，含义比谐音还直接。仓鼠腮帮子塞满食物的行为就是活生生的"收集"画面，IP化极强。仓鼠窝=你的食单，腮帮子越鼓=收录越多。仓鼠跟蚂蚁森林是一个级别的组合逻辑：蚂蚁+森林=积少成多种树，仓+鼠=粮仓里的收集者。
03 囤类 豚(tún) = 囤(tún) → 囤积收集
囤是Z世代高频词——囤货、囤券、囤皮肤、囤机票。囤=大量收集储存。豚(tún)跟囤(tún)精确同音。豚鼠=囤鼠=会囤的鼠，河豚=河囤=河里囤东西的鱼。还有吞(tūn)≈囤(tún)，吞拿鱼≈囤拿鱼。
豚鼠 类最佳 豚(tún=囤tún) + 鼠 = 囤鼠 tún shǔ
豚=囤精确同音，豚鼠=囤鼠=会囤的小鼠。豚鼠（荷兰猪）是Z世代热门宠物，圆滚滚的特别萌。虽然豚鼠实际上不怎么囤食，但名字的谐音完美——囤鼠=会囤东西的鼠。而且"豚"在日语中就是猪=美食，豚鼠自带食物基因。
画面感：一只圆滚滚的豚鼠，嘴里塞满了好东西，腮帮子鼓鼓的
精确同音（豚=囤）；Z世代热门宠物；豚=猪=美食（日语）；形象极萌
豚鼠实际不囤食（跟仓鼠不同）；豚鼠=荷兰猪可能有混淆
河豚 豚(tún=囤tún) + 河 = 河囤 hé tún
豚(=囤)+河。河豚=河囤=河里囤东西的鱼。河豚的物理特征是"膨胀"——吃满了气鼓起来，越囤越鼓。你的食单也一样，越吃越满。河豚还是一种高端美食。
画面感：河豚吃满了气，圆鼓鼓的，满足感爆棚
精确同音；膨胀=越囤越满的视觉隐喻；本身就是美食
河豚有毒的联想；可能与食品安全矛盾
海豚 豚(tún=囤tún) + 海 = 海囤 hǎi tún
豚(=囤)+海。海豚=海囤=海里囤东西的。海豚是大家最喜爱的海洋动物之一，形象亲切聪明。
画面感：海豚跃出水面，嘴里叼着一条鱼
精确同音；海豚形象极好认知度高
海豚与美食关联弱；海豚保护动物不宜食用
吞拿鱼 吞(tūn≈囤tún) + 拿 + 鱼 = 囤拿鱼 tūn ná yú
吞(≈囤)+拿+鱼。吞拿鱼(金枪鱼)=囤拿鱼=囤着拿的鱼。吞=吞下=收集进肚子，拿=获取，鱼=美食。三个字组成了"囤+拿+美食"的完整链路。而且吞拿鱼本身就是常见美食（寿司刺身）。
画面感：一条大金枪鱼，吞下满满的好味道

豚蹄 豚(tún=囤tún) + 蹄 = 囤蹄 tún tí
豚(=囤)+蹄。豚蹄=囤蹄=囤着的好蹄。猪蹄是美食，胶原蛋白满满。

豚狸 豚(tún=囤tún) + 狸(lí≈理lǐ) = 囤理 tún lí
豚(=囤)+狸(≈理)。豚狸=囤理=囤+整理=收集并整理。豚狸是豚鼠的古称。收集+整理=策展。
画面感：一只小豚狸在认真整理它收集的好东西
囤+理=收集+策展双含义

04 录类 鹿/鹭(lù) = 录(lù) → 记录收录
录=记录、收录、登录。你的食单就是你的美食"录"。鹿(lù)和鹭(lù)跟录(lù)精确同音。梅花鹿=美画录=美丽画卷的记录，白鹭=白录=空白记录等你填满。长颈鹿=长景录=大视野的记录。
白鹭 类最佳 鹭(lù=录lù) + 白 = 白录 bái lù
白鹭=白录=空白记录，等你用美食填满。鹭(=录)精确同音。白鹭=白录=一张白纸的记录——你的食单刚创建时是空白的，随着你消费一家家店，白录逐渐被填满。白鹭是极美的鸟，站在水边等待猎物的姿态=你等待发现好店的姿态。"两个黄鹂鸣翠柳，一行白鹭上青天"——白鹭有天然的诗意。
画面感：一只白鹭站在水边，洁白如纸，等待鱼游过来——等待被填满的空白记录
精确同音（鹭=录）；"白"=空白=等待填满的完美隐喻；白鹭形象极美有诗意；品牌画面感强

鹿茸 鹿(lù=录lù) + 茸(róng≈容róng) = 录容 lù róng
鹿(=录)+茸(≈容)。鹿茸=录容=收录内容=记录的内容。你的食单里收录的所有内容。鹿茸是名贵食材。
画面感：毛茸茸的鹿茸，柔软又珍贵
录+容双谐音；鹿茸是名贵食材
茸/容声调不同；鹿茸偏药材
水鹿 鹿(lù=录lù) + 水 = 水录 shuǐ lù
鹿(=录)+水。水鹿=水录=流水记录。你的消费记录像流水一样不断积累。水鹿是南方常见的鹿种。
画面感：水鹿在溪边饮水，流水潺潺
"流水录"=消费记录的隐喻
水鹿知名度低；与美食无关联
牛背鹭 鹭(lù=录lù) + 牛背 = 牛背录 niú bèi lù
鹭(=录)+牛背。牛背鹭=牛背录=在牛背上记录。牛背鹭站在牛背上吃虫子=共生关系=你跟好店的共生关系。
画面感：牛背鹭站在水牛背上，悠闲自在
有共生关系的隐喻
三字偏长；知名度极低
本类精选：白鹭。鹭=录精确同音，白=空白=等待填满的记录。白鹭形象极美有诗意，"白录"=一张空白记录等你用美食填满。梅花鹿的"美画录"和斑点标记隐喻也很巧妙，但三字偏长。
05 夹类 荚(jiá) ≈ 夹(jiā) → 收藏夹
夹=夹子、文件夹、收藏夹——你的浏览器收藏夹就是"夹"。荚(jiá)跟夹(jiā)近音。豆荚=豆夹=食物+收藏夹。豆荚是一个壳里面夹着一排豆子，打开就是一排整齐的收藏——天然就是"收藏夹"的物理形态。
豆荚 类最佳 荚(jiá≈夹jiā) + 豆 = 豆夹 dòu jiá
豆=食物，荚=收藏夹，豆荚=食物收藏夹。荚(jiá)≈夹(jiā)，豆荚=豆夹=装豆子的夹子=你的美食收藏夹。豆荚的物理形态完美对应收藏夹——一个壳里面夹着一排豆子，打开就是一排整齐排列的条目。每颗豆子=你收录的一家好店，整个豆荚=你的食单。豆荚还是植物、是食物、是容器，三合一。
画面感：掰开一个豆荚，里面一排豆子整整齐齐排列，每颗是一家好店
豆=食物+荚=收藏夹（双重含义）；豆荚物理形态=收藏夹（天然对应）；是植物是食物是容器三合一；画面感极强
荚/夹声调不同（jiá vs jiā）；荚字偏书面
皂荚 荚(jiá≈夹jiā) + 皂(zào≈造zào) = 造夹 zào jiá
荚(≈夹)+皂(≈造)。皂荚=造夹=创造收藏夹=创建你的食单。皂荚是天然植物，古人用来洗衣服。
画面感：皂荚树上挂着一串串皂荚
造+夹双含义（创建收藏夹）
皂荚偏清洁用品不偏美食；皂字认知偏差
槐荚 荚(jiá≈夹jiā) + 槐(huái≈怀huái) = 怀夹 huái jiá
荚(≈夹)+槐(≈怀)。槐荚=怀夹=怀抱收藏夹=把好店揣在怀里。槐树是北方常见行道树。
画面感：槐树下捡起一串槐荚，揣进口袋
怀+夹双含义（怀抱收藏）
槐荚知名度低；与美食无关联
花生荚 荚(jiá≈夹jiā) + 花生 = 花生夹 huā shēng jiá
荚(≈夹)+花生。花生荚=花生夹=装花生的收藏夹。花生本身是美食，花生壳=收藏容器。一颗花生=一家好店。
画面感：剥开花生壳，里面躺着两颗花生仁
花生=美食+荚=收藏夹；花生壳=容器
三字偏长；花生荚不如豆荚通用
榆钱 钱(qián≈签qiān) + 榆 = 榆签 yú qián
钱(≈签)+榆。榆钱=榆签=榆树的标签。榆钱是榆树的翅果，圆圆的像铜钱串成一串。签=书签=标签=收录标记。一串榆钱=一串好店的标签。
画面感：榆树上挂满一串串圆圆的榆钱，像一串书签
榆钱串=收藏串的隐喻；钱/签近音；榆钱本身可食用
钱/签声调不同；榆钱知名度偏低
合欢 合(hé=盒hé) + 欢 = 盒欢 hé huān
合(=盒)+欢。合欢=盒欢=快乐的盒子=装满好东西的美食盒。合欢花是粉色的绒花，极美。合=合拢=收拢=收集。
画面感：合欢树上粉色的绒花，像一盒盒粉色棉花糖
合=盒精确同音；合欢花极美；"快乐盒子"有温度
合欢偏观赏不偏美食；与收集关联需解释
夹竹桃 夹(jiā) + 竹 + 桃 = 夹竹桃 jiā zhú táo
夹(=收藏夹)直接出现在名字里。夹竹桃=收藏竹子和桃子的夹子=收集美食(桃)的夹子。
画面感：夹竹桃开着粉花
夹字直接出现
夹竹桃有毒；四字太长；与美食无关
豆角 角(jiǎo≈夹jiā) + 豆 = 豆夹 dòu jiǎo
角(≈夹)+豆。豆角=豆夹=装豆子的夹子。跟豆荚类似的逻辑，但豆角更口语化、更日常。
画面感：菜摊上一把把青翠的豆角
豆角=日常美食；口语化
角/夹近音较远；豆角太日常不够品牌化
本类精选：豆荚。豆=食物+荚=收藏夹，一个名字同时命中美食和收集两个维度。豆荚的物理形态——一个壳里夹着一排豆子——完美对应收藏夹的形态：一个列表里排列着一条条收录。每颗豆子=一家好店，整个豆荚=你的食单。
06 聚类 橘(jú) ≈ 聚(jù) → 聚集收集
聚=聚集、汇聚、聚在一起。你的食单=你聚集的好店。橘(jú)跟聚(jù)近音（韵母相同，声母j相同，仅ü/u差异）。金橘=金聚=金色聚集，橘子=聚子=聚集的东西。橘是水果，天然美食，且橘色明亮年轻。
金橘 类最佳 橘(jú≈聚jù) + 金 = 金聚 jīn jú
金=珍贵，橘=聚，金橘=金聚=珍贵的聚集。橘(jú)≈聚(jù)，金橘=金聚=金色聚集=珍贵的收集。你的食单=你珍贵的美食聚集。金橘本身就是一种小巧可爱的水果，一口一个，像你食单上每一条精挑细选的收录。金橘还谐音"金句"——你的食单里每一家都是"金句"级的好店。
画面感：一盆金橘盆栽，挂满金色小果子，每颗是一家好店
橘≈聚近音；金=珍贵+橘=聚集；金橘本身小巧可爱；金橘谐音金句；橘色年轻明亮
橘/聚非精确同音（jú vs jù）；金橘体积小可能显轻
橘子 橘(jú≈聚jù) + 子 = 聚子 jú zi
橘(≈聚)+子。橘子=聚子=聚集的东西。最日常的水果，认知度最高。
画面感：剥开一个橘子，一瓣一瓣分着吃
认知度最高；橘瓣=条目的隐喻
太普通不够品牌化；非精确谐音
蜜橘 橘(jú≈聚jù) + 蜜 = 蜜聚 mì jú
橘(≈聚)+蜜。蜜橘=蜜聚=甜蜜聚集。蜜=美食/甜蜜，橘=聚集。双含义：甜蜜的美食+聚集好店。
画面感：蜜橘汁水丰盈，甜蜜满溢
蜜=美食+橘=聚集双命中
非精确谐音；蜜橘偏品种名
丑橘 橘(jú≈聚jù) + 丑 = 丑聚 chǒu jú
橘(≈聚)+丑。丑橘=丑聚=丑萌的聚集。丑橘外表皱巴巴但内心超甜=你的食单不靠颜值靠真材实料。Z世代喜欢"丑萌"。
画面感：一个皱巴巴的丑橘，掰开超甜
丑萌=Z世代审美；"外表丑内心甜"=真材实料的隐喻
丑字部分人不喜欢；非精确谐音
砂糖橘 橘(jú≈聚jù) + 砂糖 = 砂糖聚 shā táng jú
橘(≈聚)+砂糖。砂糖橘=砂糖聚=甜蜜聚集。砂糖橘是过年必备水果，有团聚感。
画面感：过年一家人围着吃砂糖橘，其乐融融
有团聚感；认知度高
三字偏长；非精确谐音
橘红 橘(jú≈聚jù) + 红 = 聚红 jú hóng
橘(≈聚)+红。橘红=聚红=聚集红火=越聚越红。你的食单越来越红火。橘红是一种药材也是颜色。
画面感：橘红色暖洋洋的，像夕阳
有"越聚越红"的兴旺感
橘红偏颜色/药材；非精确谐音
橘柚 橘(jú≈聚jù) + 柚(yòu≈有yǒu) = 聚有 jú yòu
橘(≈聚)+柚(≈有)。橘柚=聚有=聚集拥有的=你聚集的好东西。橘柚是一种柑橘类水果。
画面感：一个橘柚，外皮绿内心橙
聚+有双含义
柚/有声调不同；橘柚知名度低
枸橘 橘(jú≈聚jù) + 枸(gǒu≈够gòu) = 够聚 gǒu jú
橘(≈聚)+枸(≈够)。枸橘=够聚=够多的聚集=收集够了=食单够丰富。枸橘是中药枳实的来源。
画面感：枸橘树上结满小果子
够+聚双含义有趣
枸/够声调不同；枸橘知名度极低
本类精选：金橘。金=珍贵+橘(≈聚)=聚集，金橘=金聚=珍贵的聚集。金橘本身小巧可爱，谐音金句（每家店都是金句），橘色年轻明亮。丑橘的丑萌反差感也很有Z世代味道，但金橘更品牌化。
07 满类 鳗(mán) / 蔓(màn) ≈ 满(mǎn) → 收集满了
满=收集满了、丰富、充实。你的食单越收集越满。鳗(mán)和蔓(màn)跟满(mǎn)近音。鳗鱼=满鱼=越来越满的鱼，蔓越莓=满越莓=越来越满的莓。满天星=满=直接就是"满"字，一盆花里密密麻麻全是小花。
满天星 类最佳 满(mǎn=满mǎn) + 天星 = 满天星 mǎn tiān xīng
满字直接出现在名字里，满天星本身就是"收集满了"的视觉隐喻。满天星是一种花，一根枝上密密麻麻开满无数小花——就像你的食单上密密麻麻收录了一家又一家好店。每一朵小花=你收录的一家店，整盆满天星=你的食单。"满天星"三个字就是"收集满了"的画面——满天都是星星，全是你发现的好店。而且满天星是Z世代花艺常见花材，年轻女性极熟悉。
画面感：一盆满天星，密密麻麻全是白色小花，每朵是你收录的一家好店
满=收集满了（直接命中）；满天星物理形态=密密麻麻的收集（每朵花=每家店）；Z世代花艺常见花材；画面感极美
三字偏长；满天星偏花艺与美食无直接关联
鳗鱼 鳗(mán≈满mǎn) + 鱼 = 满鱼 mán yú
鳗(≈满)+鱼。鳗鱼=满鱼=越来越满的鱼。鳗鱼本身就是美食（鳗鱼饭），与美食直接关联。鳗鱼滑溜溜的、饱满的形态=食单越吃越满。
画面感：一碗鳗鱼饭，鳗鱼铺满米饭，满足感爆棚
鳗鱼=美食直接命中；鳗鱼饭=铺满=满了的视觉
鳗/满声调不同（mán vs mǎn）；鳗鱼偏日式
蔓越莓 蔓(màn≈满mǎn) + 越 + 莓 = 满越莓 màn yuè méi
蔓(≈满)+越+莓。蔓越莓=满越莓=越来越满的莓果。蔓=藤蔓（蔓延生长）+满（越来越满）+越（越来越多）+莓（美食）。三个字组成了"越来越满越多"的收集意象。蔓越莓还是Z世代健康饮品常见元素。
画面感：蔓越莓一颗颗红亮饱满，越收越多
蔓+越+莓三字全有含义（满+越来越+美食）；Z世代饮品元素
蔓/满声调不同；三字偏长
满江红 满(mǎn=满mǎn) + 江红 mǎn jiāng hóng
满字直接出现。满江红是一种水生蕨类植物，秋天水面铺满红色=满江红。你的食单=满江红=铺满好店。满江红还是岳飞的词牌名。
画面感：秋天水面铺满红色小叶，密密麻麻
满=直接命中；铺满=收集满的视觉
三字偏长；满江红偏诗词联想；与美食无关联
鳗鲡 鳗(mán≈满mǎn) + 鲡(lí≈理lǐ) = 满理 mán lí
鳗(≈满)+鲡(≈理)。鳗鲡=满理=满+整理=收集满了再整理=策展。鳗鲡是鳗鱼的古称。
画面感：一条鳗鲡在水中游，满满当当
满+理=收集+策展双含义
鲡字生僻；鳗/满声调不同
蔓菁 蔓(màn≈满mǎn) + 菁(jīng≈精jīng) = 满精 màn jīng
蔓(≈满)+菁(≈精)。蔓菁=满精=满+精选=收集满了精选的好东西。蔓菁是一种根茎蔬菜，本身就是食物。
画面感：地里拔出一根蔓菁，圆滚滚的
满+精=收集+精选双含义；本身就是食物
蔓菁知名度低；蔓/满声调不同
蔓藤 蔓(màn≈满mǎn) + 藤(téng≈腾téng) = 满腾 màn téng
蔓(≈满)+藤(≈腾)。蔓藤=满腾=满满腾腾=丰富充实。藤蔓蔓延生长=你的食单不断扩展。
画面感：藤蔓蔓延攀爬，越爬越远越满
藤蔓蔓延=扩展的隐喻
与美食无关联；蔓/满声调不同
本类精选：满天星。满字直接出现在名字里，满天星的物理形态（密密麻麻全是小花）完美对应"收集满了"的视觉——每朵小花=一家好店，整盆=你的食单。鳗鱼的美食关联更强但谐音不如满天星直接。
08 单类 丹(dān) = 单(dān) → 清单食单
单=清单、食单、菜单——你的食单就是你的"单"。丹(dān)跟单(dān)精确同音。丹桂=单桂=清单+桂花（美食），牡丹=母单=母清单。丹本身还有红色、精华的含义，视觉感强。
丹桂 类最佳 丹(dān=单dān) + 桂 = 单桂 dān guì
丹=单=清单/食单，桂=桂花=美食。丹(dān)跟单(dān)精确同音。丹桂=单桂=清单+桂花。桂花本身就是美食元素——桂花糕、桂花酒、桂花茶、桂花糖藕。你的食单=你的丹桂=你的美食清单。丹=红色=标记（你在清单上打的红勾），桂=桂花=好味道。丹桂秋天开花，满院飘香=你的食单满载好味道。
画面感：秋天丹桂飘香，金黄小花落满地——你的美食清单香气四溢
丹=单精确同音；桂花=美食元素直接命中；丹=红色=标记；丹桂飘香有画面感
丹桂偏传统花卉可能不够年轻；桂字与收集无直接关联
牡丹 丹(dān=单dān) + 牡(mǔ≈母mǔ) = 母单 mǔ dān
丹(=单)+牡(≈母)。牡丹=母单=母清单=总清单=所有食单的母版。牡丹是国花级别，"唯有牡丹真国色"。
画面感：一朵硕大的牡丹花，雍容华贵
丹=单精确同音；牡丹认知度极高
牡丹过于传统/华贵；与美食无直接关联
丹顶鹤 丹(dān=单dān) + 顶 = 单顶 dān dǐng hè
丹(=单)+顶。丹顶鹤=单顶鹤=清单顶端的鹤=清单上排名最前的=你食单的TOP1。丹顶=红顶=你的精选标记。
画面感：丹顶鹤头顶一抹红，优雅站立
"单顶"=清单TOP的隐喻巧妙
三字偏长；与美食无关联
